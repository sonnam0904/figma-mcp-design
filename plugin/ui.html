About

v1.0.0

This plugin lets MCP-compatible agents talk to Figma so AI tools can read selections, create frames, replace text, and more.

How to use

  1. Start the Go WebSocket relay on port 3055.
  2. Configure your MCP client from the MCP Config tab.
  3. Connect the plugin and copy the generated channel name.
  4. Run Figma tools from Cursor, Codex, Claude or any MCP agent.

Anonymous usage analytics (plugin opens, command names, success/error) are sent to Google Analytics to help improve the plugin. No file content or personal data is collected.
